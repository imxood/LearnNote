
# gflags

    gflags是google开源的一套命令行参数解析工具，比getopt()函数功能要强大，使用起来更加方便，gflags还支持从环境变量和配置文件中读取参数。目前有C++和Python版本

    sudo apt-get install libgflags-dev -y

    tutorial: http://blog.csdn.net/achelloworld/article/details/41959595
